# ShopExpress - Интернет-магазин на Django

Полнофункциональный интернет-магазин с дизайном в стиле AliExpress.

## Возможности

### Основной функционал
- Каталог товаров с категориями
- Детальная страница товара с галереей изображений и характеристиками
- Корзина покупок
- Оформление заказов с комментариями
- Поиск товаров
- Сортировка товаров (по цене, новизне, популярности)
- Фильтрация товаров (по категории, цене, наличию)
- Мультиязычность (русский, английский, узбекский)

### Главная страница
- Баннер/слайдер с акциями и новинками
- Блок "Популярные категории"
- Блок "Новые товары"
- Блок "Хиты продаж"
- Блок "Почему выбирают нас?" (преимущества)
- Карусель логотипов спонсоров/партнеров
- Блок FAQ (часто задаваемые вопросы)

### Дополнительные страницы
- Страница "О нас" с информацией о компании
- Страница "Контакты" с формой обратной связи и картой
- Отдельная страница FAQ с категориями

### Админ-панель
- Управление товарами (с характеристиками и множественными изображениями)
- Управление категориями
- Управление заказами
- Управление баннерами
- Управление спонсорами/партнерами
- Управление FAQ (категории и вопросы)
- Управление информацией о компании
- Управление преимуществами компании
- Просмотр сообщений из формы обратной связи

### Уведомления
- Email уведомления при оформлении заказа
- Telegram уведомления при оформлении заказа (опционально)

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Выполните миграции:
```bash
python manage.py makemigrations
python manage.py migrate
```

3. Создайте тестовые данные:

   **Базовые данные (категории и товары):**
   ```bash
   python manage.py create_test_data
   ```

   **Полные тестовые данные (все модели с изображениями):**
   ```bash
   python manage.py create_full_test_data
   ```
   
   Эта команда создаст:
   - Информацию о компании
   - Баннеры для главной страницы
   - Спонсоров/партнеров
   - Преимущества компании
   - FAQ (категории и вопросы)
   - Характеристики товаров
   - Дополнительные изображения товаров
   - Тестовые сообщения из формы обратной связи

4. (Опционально) Исправьте slug у существующих категорий:
```bash
python manage.py fix_slugs
```

5. Создайте суперпользователя:
```bash
python manage.py createsuperuser
```

6. Запустите сервер:
```bash
python manage.py runserver
```

Или используйте `run.bat` для быстрого запуска.

7. Откройте в браузере:
- Сайт: http://127.0.0.1:8000/
- Админ-панель: http://127.0.0.1:8000/admin/

## Структура проекта

- `store/` - основное приложение магазина
  - `models.py` - модели данных (Category, Product, Cart, Order и т.д.)
  - `views.py` - представления (views)
  - `urls.py` - маршруты URL
  - `admin.py` - настройка админ-панели
- `templates/store/` - HTML шаблоны
- `static/` - статические файлы (CSS, JS)
- `media/` - загружаемые файлы (изображения товаров)

## Использование

После создания тестовых данных у вас будут:
- 8 основных категорий с подкатегориями
- 20+ тестовых товаров с описаниями, ценами и рейтингами

Вы можете:
1. Войти в админ-панель для управления товарами
2. Добавить изображения товаров через админ-панель
3. Просматривать товары на сайте
4. Добавлять товары в корзину
5. Оформлять заказы

**Примечание:** Тестовые товары создаются без изображений. Загрузите изображения через админ-панель для полного функционала.

## Настройка

### 1. Базовые настройки

После установки зависимостей и выполнения миграций создайте суперпользователя:

```bash
python manage.py createsuperuser
```

### 2. Настройка информации о компании

1. Войдите в админ-панель: http://127.0.0.1:8000/admin/
2. Перейдите в раздел "Информация о компании"
3. Заполните информацию о компании, контакты, социальные сети
4. При необходимости добавьте координаты для карты (широта и долгота)

### 3. Настройка Email уведомлений

В файле `shop/settings.py` настройте параметры email:

```python
EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
EMAIL_HOST = 'smtp.gmail.com'
EMAIL_PORT = 587
EMAIL_USE_TLS = True
EMAIL_HOST_USER = 'your-email@gmail.com'
EMAIL_HOST_PASSWORD = 'your-password'
DEFAULT_FROM_EMAIL = 'your-email@gmail.com'
```

### 4. Настройка Telegram уведомлений (опционально)

1. Создайте бота через [@BotFather](https://t.me/BotFather) в Telegram
2. Получите токен бота
3. Узнайте ID чата (можно использовать [@userinfobot](https://t.me/userinfobot) или создать группу)
4. В файле `shop/settings.py` добавьте:

```python
TELEGRAM_BOT_TOKEN = 'your-bot-token'
TELEGRAM_CHAT_ID = 'your-chat-id'
```

### 5. Настройка Google Maps (опционально)

1. Получите API ключ в [Google Cloud Console](https://console.cloud.google.com/)
2. В файле `templates/store/contact.html` замените `YOUR_API_KEY` на ваш ключ

### 6. Заполнение контента

После настройки базовой информации:

1. **Баннеры**: Добавьте баннеры для главной страницы в разделе "Баннеры"
2. **Спонсоры**: Добавьте логотипы партнеров в разделе "Спонсоры"
3. **Преимущества**: Добавьте преимущества компании в разделе "Преимущества"
4. **FAQ**: Создайте категории и вопросы в разделах "Категории FAQ" и "FAQ"
5. **Товары**: Добавьте товары с характеристиками и изображениями
